# JavaScript - план лекций

Этот репозиторий содержит материалы и практику по JavaScript (после блока HTML/CSS). План построен по логике: **основа языка → работа в браузере (DOM/Events) → асинхронность и API → модули и инструменты → тестирование и деплой**.

## План (16 лекций)

### 1) Лекция 1 (как у вас)

Введение + запуск + console + переменные + типы + операции + prompt/alert/confirm + преобразования + ==/=== (база)

### 2) Лекция 2 (условия и логика - довести до конца)

boolean, сравнения (короткое закрепление), truthy/falsy, if/else/else if, тернарный, && || !, проверки ввода, switch

### 3) Циклы: while / do...while / for

паттерны: “повторяй пока корректно”, сумма, счётчики, поиск, break/continue

### 4) Функции (база)

объявление/выражение, параметры, return, scope на базовом уровне
(без замыканий, как вы и хотите)

### 5) Функции + практикум на задачах

рефакторинг задач 2-4: выносим проверки и расчёты в функции

### 6) Массивы: база + перебор

индексы, базовые методы, for и for...of

### 7) Массивы: методы + задачи

map/filter/find/some/every/reduce - ровно столько, сколько нужно для практики

### 8) Объекты: свойства/методы + перебор

объекты, массив объектов, for...in, Object.keys/values/entries

### 9) ООП: прототипы (коротко) → классы

что такое прототип, зачем знать; class, constructor, методы

### 10) ООП: наследование + практика

extends/super, простая и понятная модель (например User/Admin или Product/DigitalProduct)

### 11) DOM: основы (без Vite)

поиск, изменение, создание/удаление; рендер списка из массива объектов

### 12) События + формы/валидация

input/submit/click, preventDefault, валидация, сообщения об ошибках, делегирование (опционально)

### 13) BOM минимально + storage

таймеры, location (по минимуму), localStorage/sessionStorage + JSON, сохранение состояния

### 4) Async: callbacks → promises → async/await + fetch

fetch, try/catch, статусы, обработка ошибок, рендер данных с API

### 15) Modules + Vite + npm (tooling)

ES modules, структура проекта, dev/build, подключение библиотек

### 16) Тестирование (Vitest) + Deploy/production/оптимизация

тесты на чистые функции + деплой (1 понятный сценарий) + чеклист продакшена
